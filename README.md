# Install
## Gradle Groovy
```
repositories {
    maven {
        url = uri("https://maven.pkg.github.com/CronBotLibrary/JDALib")
    }
}
dependencies {
    implementation 'jp.cron:jdalib:0.1'
}
```
## Kotlin DSL
```
repositories {
    maven {
        url = uri("https://maven.pkg.github.com/CronBotLibrary/JDALib")
    }
}
dependencies {
    implementation("jp.cron:jdalib:0.1")
}
```
## Maven
```
<dependency>
  <groupId>jp.cron</groupId>
  <artifactId>jdalib</artifactId>
  <version>0.1</version>
</dependency>
```
